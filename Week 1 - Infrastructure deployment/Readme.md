**# Week 1 – SIEM Infrastructure Deployment**



\## Objective

Set up the core SIEM infrastructure using Wazuh and connect a Windows endpoint for security monitoring.



\## Environment Setup

\- Wazuh Manager deployed on Ubuntu server

\- Wazuh Indexer configured for log storage

\- Wazuh Dashboard installed for visualization

\- Windows 10 endpoint configured as monitored system



\## Tasks Completed

\- Installed and configured Wazuh Manager

\- Set up Wazuh Dashboard for security event visualization

\- Installed Wazuh Indexer for log indexing

\- Deployed Wazuh agent on Windows endpoint

\- Installed Sysmon to enhance Windows event logging

\- Verified agent connectivity with the Wazuh server



\## Outcome

The SIEM infrastructure was successfully deployed and the Windows system began sending system and security logs to the Wazuh server.



\## Skills Demonstrated

\- SIEM deployment

\- Endpoint monitoring setup

\- Security log collection

\- Sysmon integration
